# TMUX Setup Instructions

## 1. Install TPM (already done ✅)
TPM has been cloned to `~/.tmux/plugins/tpm`

## 2. Install Plugins
1. Start tmux: `tmux`
2. Press `prefix + Shift+i` (Ctrl+a, luego suelta y presiona Shift+i) to install plugins
3. Wait for installation to complete

## 3. Reload Configuration
If tmux is already running:
```bash
tmux source ~/.tmux.conf
```
Or press `prefix + r` (Ctrl+a, then r)

## 4. Verify vim-tmux-navigator
The plugin is already in your Neovim config. Test it:
1. Open Neovim in a tmux pane
2. Split the pane: `prefix + |` or `prefix + -`
3. Navigate with `Ctrl+h/j/k/l` - should work seamlessly between vim and tmux

## New Features

### Navigation
- `Ctrl+h/j/k/l` - Navigate between panes (works in vim AND tmux)
- `Ctrl+\` - Go to previous pane

### Splits
- `prefix + |` - Vertical split
- `prefix + -` - Horizontal split
- `prefix + v` - Vertical split (alternative)
- `prefix + s` - Horizontal split (alternative)

### Sessions
- `prefix + S` - Session switcher
- `prefix + Ctrl+c` - Create new session
- Auto-restore on tmux start (continuum)
- Auto-save every 15 minutes

### Copy Mode
- `prefix + Space` - Enter copy mode
- `v` - Start selection
- `y` - Copy to clipboard (auto-copies to system)
- Mouse drag also copies to clipboard

### Other
- `prefix + Ctrl+s` - Toggle pane sync (run same command in all panes)
- `prefix + b` - Break pane to new window
- `prefix + r` - Reload config
- Mouse support enabled

## Backup
Your old config is saved at: `~/.tmux.conf.backup`
